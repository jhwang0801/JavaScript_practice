## 변수 Variables
- 값을 저장하는 공간
- 자료를 저장할 수 있는 **이름이 주어진 기억장소**

### let
- 재할당(Reassignable) 가능
- 변경(Mutable): YES


### const
- 재할당(Reassignable) 불가능
- 변경(Mutable): YES
    - 이유: 메모리셀 안에 있는 데이터 셀을 변경하는 것이 아니라 가리키고 있는 특정한 Object 내부를 수정하는 것이므로 변경이 가능!!
- 한번 할당하고 그 이후로도 재할당이 불필요한 경우에 사용
- 프로그래밍할 때 왠만하면 let 보다 const를 선호해서 작성하는 것이 좋다!
